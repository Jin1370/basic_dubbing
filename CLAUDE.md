## 하네스: 모바일 더빙 앱

**목표:** Perso AI API를 활용한 모바일 더빙 앱을 와이어프레임부터 배포까지 에이전트 팀으로 개발한다.

**트리거:** 더빙 앱 개발, 앱 구현, 프로젝트 시작, 화면 설계, 프론트엔드/백엔드/QA 관련 작업 요청 시 `dubbing-orchestrator` 스킬을 사용하라. 단순 질문은 직접 응답 가능.

**기술 스택:**
- 프론트엔드: Expo (React Native + TypeScript)
- 백엔드: FastAPI (Python)
- 외부 API: Perso AI API
- 인프라: Supabase (Auth + Database + Storage)
