# 모바일 더빙 앱 디자인 스킬

## 대상 앱

Perso AI API를 활용한 모바일 더빙 앱. 사용자가 영상을 업로드하면 AI가 다른 언어로 더빙해주는 서비스.

## 핵심 화면 목록

1. **온보딩/로그인** — Google 로그인 (Supabase Auth)
2. **홈** — 더빙 이력 목록, 새 더빙 시작 버튼
3. **영상 업로드** — 갤러리에서 영상 선택 또는 촬영
4. **더빙 설정** — 원본 언어/타겟 언어 선택
5. **더빙 진행** — 프로그레스 표시, 예상 시간
6. **결과 확인** — 원본/더빙 비교 재생, 다운로드

## 와이어프레임 작성 규칙

ASCII 기반으로 작성한다. 각 화면마다 다음을 포함:
- 화면 이름과 목적
- 주요 UI 요소 배치 (헤더, 버튼, 입력, 리스트 등)
- 터치 타겟은 [버튼명] 형태로 표시
- 화면 전환 화살표로 플로우 표시

## 디자인 스펙 항목

| 항목 | 내용 |
|------|------|
| 색상 | Primary, Secondary, Background, Text, Error, Success |
| 타이포 | 제목(H1~H3), 본문, 캡션 크기/무게 |
| 간격 | 기본 패딩, 마진, 컴포넌트 간 갭 |
| 컴포넌트 | 버튼, 카드, 입력, 모달, 프로그레스 바 스타일 |
| 네비게이션 | 탭 구조 또는 스택 구조, 전환 애니메이션 |
