# 모바일 더빙 앱 — 와이어프레임

---

## 1. 온보딩/로그인

**목적:** 사용자가 Google 계정으로 간편하게 로그인하여 서비스에 진입한다.

**전환:** 로그인 성공 -> 홈 화면

```
+----------------------------------+
|                                  |
|           (상단 여백 80pt)        |
|                                  |
|         +-----------------+      |
|         |                 |      |
|         |   앱 로고/아이콘  |      |
|         |   (120x120pt)   |      |
|         |                 |      |
|         +-----------------+      |
|                                  |
|          하네스 더빙 스튜디오       |
|         (앱 이름, H1, 중앙정렬)    |
|                                  |
|     AI로 영상을 다른 언어로 더빙    |
|        (서브텍스트, Body)         |
|                                  |
|           (여백 40pt)            |
|                                  |
|   +----------------------------+ |
|   | [G] Google로 계속하기       | |
|   |     (48pt 높이, 전체 너비)   | |
|   +----------------------------+ |
|                                  |
|           (여백 16pt)            |
|                                  |
|    계속하면 이용약관에 동의합니다    |
|       (Caption, 중앙, 링크)      |
|                                  |
+----------------------------------+
```

**UI 요소:**
- 앱 로고: 120x120pt, 중앙 배치
- 앱 이름: H1, 중앙 정렬
- 설명 텍스트: Body, 중앙 정렬
- Google 로그인 버튼: 높이 48pt, 좌우 패딩 24pt, 터치 영역 48pt 이상
- 이용약관 링크: Caption, 하단 고정 아님 (스크롤 내 배치)

## 2. 홈

**목적:** 과거 더빙 이력을 확인하고, 새 더빙 작업을 시작한다.

**전환:** "새 더빙" FAB 탭 -> 영상 업로드 화면 / 이력 항목 탭 -> 결과 확인 화면

```
+----------------------------------+
| 하네스 더빙              [프로필] |
|  (H2)                  (32pt)   |
+----------------------------------+
|                                  |
|  내 더빙 목록  (H3, 좌측정렬)     |
|                                  |
|  +----------------------------+  |
|  | [썸네일]  제목_영상1.mp4    |  |
|  | (60x60)  한국어 -> 영어     |  |
|  |          2026.04.12  완료   |  |
|  +----------------------------+  |
|           (간격 12pt)            |
|  +----------------------------+  |
|  | [썸네일]  제목_영상2.mp4    |  |
|  | (60x60)  한국어 -> 일본어   |  |
|  |          2026.04.10  완료   |  |
|  +----------------------------+  |
|           (간격 12pt)            |
|  +----------------------------+  |
|  | [썸네일]  제목_영상3.mp4    |  |
|  | (60x60)  영어 -> 한국어     |  |
|  |          2026.04.08  처리중  |  |
|  +----------------------------+  |
|                                  |
|  (목록 없을 때: 빈 상태 일러스트) |
|  "아직 더빙 이력이 없어요"        |
|  "아래 버튼으로 시작해보세요"      |
|                                  |
|                    +----------+  |
|                    | + 새 더빙 |  |
|                    | (FAB,56pt)|  |
|                    +----------+  |
+----------------------------------+
```

**UI 요소:**
- 상단 바: 앱 이름(H2) + 프로필 아바타(32pt, 우측)
- 더빙 이력 리스트: FlatList, 각 항목은 카드 형태
  - 썸네일: 60x60pt, 좌측
  - 파일명: Body Bold
  - 언어 방향: Caption
  - 날짜 + 상태 배지: Caption
- 빈 상태: 중앙 일러스트 + 안내 텍스트
- FAB (Floating Action Button): 56pt, 우하단 고정, 엄지 도달 영역

## 3. 영상 업로드

**목적:** 갤러리에서 더빙할 영상을 선택한다.

**전환:** 영상 선택 완료 -> 더빙 설정 화면 / 뒤로 -> 홈

```
+----------------------------------+
| <  영상 선택                     |
|  (뒤로)    (H2)                  |
+----------------------------------+
|                                  |
|  +----------------------------+  |
|  |                            |  |
|  |                            |  |
|  |     [갤러리 접근 영역]      |  |
|  |                            |  |
|  |   +--------------------+   |  |
|  |   |                    |   |  |
|  |   |   영상 아이콘(48pt) |   |  |
|  |   |                    |   |  |
|  |   +--------------------+   |  |
|  |                            |  |
|  |   갤러리에서 영상을 선택    |  |
|  |      하세요 (Body)         |  |
|  |                            |  |
|  |   +--------------------+   |  |
|  |   | 갤러리에서 선택하기  |   |  |
|  |   |   (버튼, 48pt)     |   |  |
|  |   +--------------------+   |  |
|  |                            |  |
|  +----------------------------+  |
|                                  |
|  지원 형식: MP4, MOV (최대 500MB)|
|       (Caption, 중앙)            |
|                                  |
|  --- 영상 선택 후 ---            |
|                                  |
|  +----------------------------+  |
|  | [영상 미리보기]             |  |
|  | (16:9 비율, 전체 너비)      |  |
|  |         [재생 아이콘]       |  |
|  +----------------------------+  |
|  파일명.mp4   12.5MB   01:30    |
|  (Body)      (Caption) (Caption)|
|                                  |
|  +----------------------------+  |
|  |        다음 단계            |  |
|  |    (Primary 버튼, 48pt)    |  |
|  +----------------------------+  |
+----------------------------------+
```

**UI 요소:**
- 헤더: 뒤로 버튼(44x44pt 터치 영역) + 제목(H2)
- 업로드 영역: 점선 테두리 카드, 중앙 아이콘 + 텍스트 + 버튼
- 갤러리 선택 버튼: Primary 스타일, 48pt 높이
- 파일 제한 안내: Caption, 하단
- 선택 후 미리보기: 16:9 비율 영상 프리뷰
- 파일 정보: 파일명, 크기, 길이
- "다음 단계" 버튼: Primary, 하단 고정, 48pt 높이

## 4. 더빙 설정

**목적:** 원본 언어와 타겟 언어를 선택한다.

**전환:** "더빙 시작" 탭 -> 더빙 진행 화면 / 뒤로 -> 영상 업로드

```
+----------------------------------+
| <  더빙 설정                     |
|  (뒤로)    (H2)                  |
+----------------------------------+
|                                  |
|  [영상 썸네일 소형] 파일명.mp4   |
|  (40x40pt)         01:30        |
|                                  |
|  --------- 구분선 ----------    |
|                                  |
|  원본 언어 (Body Bold)           |
|  +----------------------------+  |
|  | 한국어                  [v]|  |
|  | (드롭다운, 48pt 높이)      |  |
|  +----------------------------+  |
|                                  |
|        [swap 아이콘, 44pt]       |
|                                  |
|  타겟 언어 (Body Bold)           |
|  +----------------------------+  |
|  | 영어                   [v]|  |
|  | (드롭다운, 48pt 높이)      |  |
|  +----------------------------+  |
|                                  |
|  지원 언어: 한국어, 영어, 일본어, |
|  중국어, 스페인어 (Caption)      |
|                                  |
|           (여백)                 |
|                                  |
|  +----------------------------+  |
|  |       더빙 시작             |  |
|  |   (Primary 버튼, 48pt)     |  |
|  +----------------------------+  |
|                                  |
+----------------------------------+
```

**UI 요소:**
- 헤더: 뒤로 버튼(44x44pt) + 제목(H2)
- 선택된 영상 요약: 소형 썸네일(40x40pt) + 파일명 + 길이
- 원본 언어 드롭다운: 48pt 높이, 전체 너비, 터치 영역 충분
- 언어 교환(swap) 아이콘: 44x44pt, 중앙 배치
- 타겟 언어 드롭다운: 48pt 높이, 전체 너비
- 지원 언어 안내: Caption
- "더빙 시작" 버튼: Primary, 하단, 48pt 높이

## 5. 더빙 진행

**목적:** 더빙 처리 상태를 실시간으로 보여준다. 사용자가 진행 상황을 파악하고 기다릴 수 있게 한다.

**전환:** 완료 -> 결과 확인 화면 / 백그라운드로 전환 시 홈에서 상태 확인 가능

```
+----------------------------------+
|  더빙 중...                      |
|  (H2)                            |
+----------------------------------+
|                                  |
|           (여백 60pt)            |
|                                  |
|       +------------------+       |
|       |                  |       |
|       | [애니메이션 아이콘] |       |
|       |   (80x80pt)     |       |
|       |                  |       |
|       +------------------+       |
|                                  |
|       AI가 더빙을 생성 중입니다   |
|          (Body, 중앙)            |
|                                  |
|           (여백 32pt)            |
|                                  |
|  현재 단계: 음성 분석 중 (2/4)   |
|       (Body Bold, 중앙)          |
|                                  |
|  +----------------------------+  |
|  |[=============             ]|  |
|  | (프로그레스 바, 8pt 높이)   |  |
|  +----------------------------+  |
|          65% (Body, 중앙)        |
|                                  |
|  단계 상세:                      |
|  [v] 1. 영상 업로드 완료         |
|  [v] 2. 음성 추출 완료           |
|  [>] 3. 음성 분석 중...          |
|  [ ] 4. 더빙 합성                |
|       (Caption, 좌측 정렬)       |
|                                  |
|  예상 남은 시간: 약 2분 30초      |
|       (Caption, 중앙)            |
|                                  |
|  +----------------------------+  |
|  |     백그라운드로 이동        |  |
|  |  (Secondary 버튼, 48pt)    |  |
|  +----------------------------+  |
+----------------------------------+
```

**UI 요소:**
- 헤더: 제목(H2), 뒤로 버튼 없음 (진행 중 이탈 방지, 백그라운드 버튼으로 대체)
- 애니메이션 아이콘: 80x80pt, 로딩 애니메이션(Lottie)
- 상태 텍스트: Body, 현재 무엇을 하고 있는지 설명
- 현재 단계: Body Bold, "음성 분석 중 (2/4)" 형태
- 프로그레스 바: 높이 8pt, 전체 너비, 라운드 끝, Primary 색상
- 퍼센트 표시: Body, 프로그레스 바 아래
- 단계 상세 목록: 체크/진행/대기 아이콘 + Caption 텍스트
- 예상 시간: Caption, 중앙
- "백그라운드로 이동" 버튼: Secondary 스타일, 48pt

## 6. 결과 확인

**목적:** 원본과 더빙된 영상을 비교 재생하고, 결과물을 다운로드한다.

**전환:** "새 더빙" -> 영상 업로드 / "홈으로" -> 홈 / "다운로드" -> 기기 저장

```
+----------------------------------+
| <  더빙 결과                     |
|  (뒤로)    (H2)                  |
+----------------------------------+
|                                  |
|  +----------------------------+  |
|  |                            |  |
|  |    [영상 플레이어 영역]     |  |
|  |    (16:9 비율, 전체 너비)   |  |
|  |                            |  |
|  |       [재생/일시정지]       |  |
|  |                            |  |
|  +----------------------------+  |
|  [====>                ] 01:30  |
|  (시크바)              (시간)   |
|                                  |
|  +------------+  +------------+  |
|  |   원본     |  |   더빙     |  |
|  | (탭, 활성) |  | (탭, 비활성)|  |
|  +------------+  +------------+  |
|                                  |
|  정보:                           |
|  원본 언어:  한국어               |
|  타겟 언어:  영어                 |
|  파일 크기:  15.2MB              |
|  더빙 완료:  2026.04.13 14:30   |
|  (Caption, 좌측 정렬)            |
|                                  |
|  +----------------------------+  |
|  |   [다운로드 아이콘] 다운로드  |  |
|  |   (Primary 버튼, 48pt)     |  |
|  +----------------------------+  |
|           (간격 12pt)            |
|  +----------------------------+  |
|  |       새 더빙 시작           |  |
|  |  (Secondary 버튼, 48pt)    |  |
|  +----------------------------+  |
|                                  |
+----------------------------------+
```

**UI 요소:**
- 헤더: 뒤로 버튼(44x44pt) + 제목(H2)
- 영상 플레이어: 16:9 비율, 전체 너비, 재생/일시정지 오버레이
- 시크바: 터치 영역 44pt 높이 확보
- 원본/더빙 전환 탭: 2개 탭, 각각 44pt 높이 이상
- 더빙 정보: Caption, 키-값 쌍 나열
- 다운로드 버튼: Primary, 아이콘 + 텍스트, 48pt
- 새 더빙 시작 버튼: Secondary, 48pt

## 화면 전환 흐름 요약

```
온보딩/로그인
     |
     v
    홈 (더빙 이력)
     |
     +--- [이력 항목 탭] ---> 결과 확인
     |
     +--- [+ 새 더빙 FAB] ---> 영상 업로드
                                   |
                                   v
                              더빙 설정
                                   |
                                   v
                              더빙 진행
                                   |
                                   v
                              결과 확인
                                   |
                                   +--- [다운로드]
                                   +--- [새 더빙] ---> 영상 업로드
                                   +--- [뒤로/홈] ---> 홈
```

## 공통 사항

- 모든 터치 타겟: 최소 44x44pt
- 하단 버튼: SafeArea 하단 패딩 적용
- 로딩 상태: 모든 비동기 작업에 스켈레톤 UI 또는 스피너 표시
- 에러 상태: 인라인 에러 메시지 + 재시도 버튼
- 한 손 조작: 주요 CTA(Call to Action) 버튼은 하단 배치
